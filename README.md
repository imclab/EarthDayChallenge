# EarthDayChallenge

SpaceAppsChallenge competition

# Requirements

* R 2.15.3
  - shiny
  - maps
  - ggplot2
* RStudio (optional)

# Run Application

## Development

In RStudio and located in the project root, run

    library(shiny)
    shiny::runApp("app")
    
and a web browser will launch with the running Shiny app.
